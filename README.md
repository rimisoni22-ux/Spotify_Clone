# 🎧 Spotify Clone

A Spotify-inspired **music player web application** built using **HTML, CSS, and JavaScript**.  
This project recreates the basic interface and functionality of Spotify including playing songs, navigating playlists, and controlling music playback.

---

## 🚀 Live Demo
🔗 https://spotify-clone-xi-ashy.vercel.app/

---

## 📌 Features

- 🎵 Play and Pause music
- ⏭ Next and Previous track
- 📃 Playlist display
- 🎚 Music progress bar
- 🎨 Clean Spotify-style UI
- 📱 Responsive design
- ⚡ Smooth JavaScript interactions

---

## 🛠 Tech Stack

- **HTML5** – Structure of the application  
- **CSS3** – Styling and layout  
- **JavaScript (Vanilla JS)** – Functionality and music controls
